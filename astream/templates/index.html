Notes Importantes

ℹ️ Compatibilité : D'autres lecteurs peuvent fonctionner mais n'ont pas été testés officiellement. Certains lecteurs peuvent également ne pas fonctionner.

📝 Lecteurs manquants : Si vous voyez un lecteur sur Anime-Sama qui n'apparaît pas dans l'addon, c'est soit un lecteur non supporté, soit un domaine exclu de votre configuration.

Problèmes récurrents

J'ai un problème avec mon épisode

Solution : Essayez d'abord de changer de lecteur vidéo. Chaque épisode propose généralement plusieurs lecteurs différents.

La vidéo charge lentement

Solution : Il arrive que les lecteurs vidéo soient saturés ou surchargés. Changez de lecteur pour en trouver un plus rapide.

Aucun stream disponible

Cause : Le lecteur sur Anime-Sama est cassé ou pas compatible avec notre addon.

Solution : Ouvrez une issue sur GitHub pour faire la demande d'ajout de l'épisode manuellement.

Liste de solutions diverses

  • Vérifiez si votre connexion internet est bonne (téléchargements, chargement de vidéos, mises à jour...)
  • Changez de lecteur vidéo
  • Attendre 1h puis réessayer
